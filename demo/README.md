# React Animated Code - Demo App

This is a demo application showcasing the features of the `react-animated-code` library.

## Features Demonstrated

1. **Basic Example** - Simple code transitions with built-in navigation
2. **With Filenames** - Display filenames in the code header
3. **React Component Evolution** - Watch a React component grow step by step
4. **Python with Custom Theme** - Different language and Dracula theme
5. **Without Line Numbers** - Clean code display
6. **Nord Theme** - Beautiful Nord color scheme
7. **Custom Controls** - Build your own navigation UI with controlled mode
8. **Auto-Play Mode** - Automatically cycle through code snippets

## Running Locally

First, make sure the parent library is built:

```bash
cd ..
npm install
npm run build
```

Then, install dependencies and run the demo:

```bash
cd demo
npm install
npm run dev
```

The demo will be available at `http://localhost:5174`

## Building for Production

```bash
npm run build
npm run preview
```

## Deployment

The demo is automatically deployed to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions.
